# Auth API

1. Clonar proyecto
2. Instalar dependencias
```
npm i
```
3. Clonar el **.env.template** y renombrarlo a **.env**
4. Cambiar variables de entorno
5. Levantar la BD
```
docker-compose up -d
```

6.- Ejecutar el seed
```
http://localhost:3000/api/seed
```

7. Levantar el proyecto
```
npm run start:dev
```
